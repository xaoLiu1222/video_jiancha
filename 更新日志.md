# 更新日志

## v1.1.0 (2024-11-28)

### ✨ 新特性

- **输出目录跟随输入目录** - 审核后的视频分类和报告现在保存在输入目录下，而不是固定的 `data/` 目录
  - 审核 U 盘视频时，结果直接保存在 U 盘，不占用本地硬盘空间
  - 更符合实际使用场景，方便管理不同来源的视频

### 🔧 改进

- 优化了目录结构说明
- 更新了所有相关文档

### 📝 示例

```bash
# 审核 U 盘视频
python siliconflow_review.py /media/usb/videos

# 结果会保存在：
# /media/usb/videos/approved/
# /media/usb/videos/rejected/
# /media/usb/videos/manual_review/
# /media/usb/videos/review_results.json
```
