# 更新日志

## v2.0.0 (2024-12-03)

### ✨ 重大更新

- **新增 Google Gemini 方案** - 使用 Gemini 2.0 Flash 原生视频理解，准确率更高
- **双方案架构** - Gemini（推荐）+ 硅基流动（备选）

### 🚀 Gemini 方案优势

- 原生视频理解，无需手动提取帧
- 支持最长 2 小时视频
- Google 最新多模态模型，准确率更高
- 有免费使用层

### 📝 使用方式

```bash
# Gemini 方案（推荐）
export GEMINI_API_KEY='your-api-key'
python gemini_review.py /path/to/videos

# 硅基流动方案（备选）
export SILICONFLOW_API_KEY='sk-xxx'
python siliconflow_review.py /path/to/videos
```

### 🔧 其他改进

- 精简项目文档
- 更新配置文件结构
- 修复 OpenCV 警告信息
