# 🚀 快速开始指南

## 三步完成视频审核

---

### 第一步：获取 API Key（5分钟）

1. 打开浏览器，访问：**https://siliconflow.cn**
2. 点击右上角"注册/登录"
3. 注册账号（支持手机号/邮箱）
4. 登录后，点击左侧"API 密钥"
5. 点击"新建 API 密钥"
6. 复制生成的密钥（sk-开头）

---

### 第二步：设置 API Key

打开终端，运行：

```bash
export SILICONFLOW_API_KEY='sk-你的密钥'
```

**永久保存（推荐）：**

```bash
echo 'export SILICONFLOW_API_KEY="sk-你的密钥"' >> ~/.bashrc
source ~/.bashrc
```

---

### 第三步：开始审核

```bash
cd ~/桌面/myProject/video_jiancha

# 1. 把待审核视频放到这个目录
cp /path/to/your/videos/*.mp4 ./data/to_review/

# 2. 运行审核脚本
./start_review.sh

# 3. 查看结果
cat ./data/results/siliconflow_review.json
```

## 🆘 遇到问题？

### 问题1：提示"未设置 API Key"

**解决：**
```bash
export SILICONFLOW_API_KEY='sk-你的密钥'
```

### 问题2：找不到视频文件

**解决：**
```bash
# 确保视频在正确目录
ls ./data/to_review/
```

### 问题3：API 调用失败

**解决：**
- 检查网络连接
- 确认 API Key 是否正确
- 查看免费额度是否用完
